A Flappy Bird clone featuring both manual play and AI-controlled gameplay using Model Predictive Control with Mixed Integer Programming.

## Features
- **Manual Mode**: Play the game yourself using keyboard controls
- **AI Mode**: Watch an AI play using advanced control algorithms
- Classic Flappy Bird gameplay with smooth animations
- High score tracking
- Multiple bird colors and backgrounds

## Requirements
- Python 3.7 or higher
- pygame
- mip (Mixed Integer Programming library)

## Installation

```shell
pip install -r requirements.txt
```

Or using pipenv:

```shell
pipenv install
pipenv shell
```

## Usage

Run the game:

```shell
python flappy.py
```

## Controls
- **Arrow Keys / W/S**: Navigate menu options
- **M**: Select Manual Mode
- **A**: Select AI Mode
- **Space / Up / W**: Flap (in manual mode) / Start game
- **ESC**: Quit game

## Game Modes
1. **Manual Mode**: Control the bird yourself and try to beat the high score
2. **AI Mode**: Watch the AI navigate through pipes using predictive control algorithms

Enjoy the game!
